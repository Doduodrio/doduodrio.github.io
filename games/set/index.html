PLAY SET

Welcome to the game of Set! In this game, you will compete against a bot to find sets of three cards that match. The cards match if their numbers, colors, shapes, and patterns are either all the same or all different. Good luck!



Bienvenue dans le jeu de Set ! Dans ce jeu, vous affronterez un bot en trouvant des sets de trois cartes qui se correspondent. Les cartes se correspondent si les nombres, les couleurs, les formes, et les motifs sont tous pareils ou tous différents. Bonne chance ! (Vous pouvez changer la langue en français en cliquant sur le bouton "Français".)
